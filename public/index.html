fCC Image Abstraction Layer

User stories:

  • I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  • I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  • I can get a list of the most recently submitted search strings.

Example search usage:

https://dcross-image-abstraction-layer.herokuapp.com/search/fish

Will return:

[
  {
    "url":"https://i.ytimg.com/vi/TSqCoBhL2Hw/maxresdefault.jpg",
    "snippet":"Boy Falling into Koi Fish Pond. - YouTube",
    "thumbnail":"https://encrypted-tbn0.gstatic.com/images?
    q=tbn:ANd9GcTeu1D7BFHCOGcDYxUa515iEhVMM2wU9gqvR-FyeD4dOV_hXeZAHArmEaA",
    "context":"https://www.youtube.com/watch?v=TSqCoBhL2Hw"
  },
  {...}
]
